# StoreSCP Tool v0.1.2 发布说明

发布日期：2026-04-07  
版本号：`v0.1.2`

## 版本简介

`StoreSCP Tool v0.1.2` 是在 `v0.1.1` 基础上的现场可用性增强版本。

本版本重点补齐了小屏笔记本使用体验、目录选择、日志查看和接收记录查看能力，并新增 `DIMSE` 信息展示与关键成功提示，更适合现场工程师直接拿来联调排障。

## 本版本重点改进

- 小屏笔记本界面布局优化
- 左侧操作区可折叠，右侧数据区可获得更大显示空间
- 新增接收目录与日志目录选择按钮
- 新增日志关键字筛选
- 新增日志 `DIMSE` 信息列，可区分 `C-ECHO` / `C-STORE` / Association 等关键链路
- 新增接收记录双击详情查看
- 新增关键 `DIMSE` 成功后的提示框
- 会话统计视图优化，更适合小窗口查看

## 已支持能力

- `StoreSCP` 启动与停止
- `Ping`
- TCP 端口检测
- `DICOM C-ECHO`
- 本机 `C-STORE` 自测
- 实时日志显示与本地日志落盘
- 日志关键字筛选与 `DIMSE` 信息展示
- 接收记录展示、详情查看与持久化
- 接收会话统计与持久化
- 诊断包导出
- 监听 IP 自动选择、下拉选择和刷新
- 接收目录 / 日志目录选择

## 默认参数

- `Local AE Title = RC120`
- `Listen Port = 5678`
- `Echo Calling AE = RC120`
- `Echo Called AE = CT_AE`
- `Listen IP` 首次启动时自动选择本机优先 IPv4
