# StoreSCP Tool v0.1.1 发布说明

发布日期：2026-04-04  
版本号：`v0.1.1`

## 版本简介

`StoreSCP Tool v0.1.1` 是面向医院现场联调场景的首个正式绿色版发布版本。

本版本聚焦于“可直接下载、解压、运行”的现场使用体验，支持在 Windows 笔记本上替代目标设备完成基础 DICOM 联调验证，适用于实施、售后、研发现场排障与联调测试。

## 主要能力

本版本已支持以下核心功能：

- `StoreSCP` 启动与停止
- `Ping`
- TCP 端口检测
- `DICOM C-ECHO`
- 本机 `C-STORE` 自测
- 实时日志显示与本地日志落盘
- 接收记录展示与持久化
- 接收会话统计与持久化
- 诊断包导出
- 监听 IP 自动选择、下拉选择和刷新

## 默认参数

当前默认参数如下：

- `Local AE Title = RC120`
- `Listen Port = 5678`
- `Echo Calling AE = RC120`
- `Echo Called AE = CT_AE`
- `Listen IP` 首次启动时自动选择本机优先 IPv4

## 交付形式

本版本以绿色版压缩包方式发布。

推荐使用方式：

1. 下载发布附件中的绿色版 zip
2. 解压到本地可写目录
3. 运行 `StorescpTool.App.exe`

说明：

- 请使用 Release 附件中的绿色版压缩包
- 不建议直接通过源码仓库下载后运行
- 不建议只单独拷贝 exe 文件运行

## 适用环境

建议运行环境：

- `64 位 Windows 10 / Windows 11`
- 允许本地写入配置、日志和接收文件
- 允许监听目标测试端口

`现场联调辅助工具 / MVP 正式发布版`

## 致使用者

如果你是在医院现场使用本工具，建议首次运行时先完成以下检查：

1. 确认本机 IP、AE Title、端口是否正确
2. 先做 `Ping`
3. 再做 TCP 端口检测
4. 再做 `C-ECHO`
5. 最后进行 CT 推送联调

如需问题反馈，建议同时附带：

- 操作时间
- 对端 AE / IP / Port
- 诊断包导出结果
- 关键错误截图或日志
